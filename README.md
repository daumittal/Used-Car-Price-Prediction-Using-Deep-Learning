# Used-Car-Price-Prediction-Using-Deep-Learning

In this project, our focus will be on data preprocessing, where we'll handle missing values and convert categorical variables into numerical representations for seamless processing. Following this, we'll leverage seaborn's visualization capabilities to gain insights into the data, revealing correlations between car attributes and sale prices.

Once the data is prepared, we'll build a robust deep learning model using TensorFlow/Keras. The Sequential model will consist of Dense layers with appropriate activation functions. To predict continuous sale prices, we'll utilize a linear activation function in the output layer.

Our model will be finely tuned with an optimal compilation, featuring an effective optimizer, mean squared error loss function, and mean absolute error as an evaluation metric. We'll carefully monitor the training process to detect signs of overfitting or underfitting. Subsequently, we'll assess the model's performance on testing data to gauge its accuracy, employing the R2 score as an evaluation metric
